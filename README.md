# Smart-Waste-Detection-and-Segregation-Platform
An AI-powered web application that classifies waste (plastic, paper, glass, metal, organic) using deep learning models trained on TrashNet and TACO datasets. Built with MERN stack + Flask/FastAPI, it allows users to upload images, get real-time predictions, and view disposal guidance with history tracking in MongoDB.

---

## 🌍 Problem Statement

Municipal Solid Waste (MSW) is a growing global crisis fueled by rapid urbanization, population growth, and consumerism. Current practices of mixing recyclable, biodegradable, and hazardous materials reduce recycling efficiency, overload landfills, and harm ecosystems.  

Manual waste segregation is:
- **Labor-intensive**
- **Error-prone**
- **Unsafe for sanitation workers**

➡️ There is a strong need for an **automated, scalable, and intelligent solution** to classify and manage waste in real-time.

---

## 🎯 Objectives

1. **AI-Powered Classification**  
   Develop a computer vision model capable of detecting and classifying different categories of waste (plastic, paper, glass, metal, organic, etc.) with high accuracy.

2. **User-Friendly Web Platform**  
   Design an interactive platform where users can upload/capture waste images and receive real-time classification and disposal guidance.

3. **Sustainability Impact**  
   Improve segregation efficiency, reduce manual effort, and promote sustainable waste management through community participation and awareness.

---

## 💡 Significance & Motivation

- Rapid urbanization has led to massive waste generation, most of which is dumped into landfills or incinerated without proper segregation.  
- This causes:
  - Soil and water contamination
  - Air pollution
  - Greenhouse gas emissions
  - Wastage of recyclable resources

🚀 **Why this project matters:**
- Reduces dependence on manual labor  
- Enhances segregation efficiency & worker safety  
- Supports the **circular economy** by maximizing recycling  
- Encourages **eco-friendly smart cities** and responsible waste management
